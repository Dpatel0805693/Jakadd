# R Service Testing

### Step 1: Start R Service
```bash
# From r/ directory
R -e "plumber::plumb('plumber.R')$run(host='0.0.0.0', port=8000)"
```

### Step 2: Run Tests (in new terminal)
```bash
Rscript r/tests/test-ols.R
```
